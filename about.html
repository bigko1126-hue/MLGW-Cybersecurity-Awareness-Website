This repository contains a small static site for MLGW Cybersecurity Awareness. It demonstrates how to structure simple pages and use relative links so navigation works both in the repo and when published to GitHub Pages.

Why relative links?

Relative links (like about.html) keep navigation working regardless of the hosting path. When GitHub Pages serves the site from https://bigko1126-hue.github.io/MLGW-Cybersecurity-Awareness-Website/, the links still point correctly to the sibling pages.

How to view

  1. Open the repository on GitHub and click index.html to preview the file.
  2. Or enable GitHub Pages in the repository settings (already enabled) and visit the Pages site URL shown in Settings > Pages.

If you'd like, I can also add a simple stylesheet, a README, or configure a custom index path for GitHub Pages.
